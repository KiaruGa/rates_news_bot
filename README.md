## Бот "Rate News"

Бот "Rate News" - это Telegram-бот, который предоставляет информацию о курсах валют и новостях пользователю. В данном руководстве будет описаны основные функции бота и способы их использования.

### Начало работы

Чтобы начать использовать бота "Rate News", выполните следующие шаги:

1. Найдите бота в Telegram или перейдите по предоставленной ссылке для запуска чата.
2. После того, как вы начали чат с ботом, отправьте команду "/start" для инициализации работы бота.

### Функциональность

Бот "Rate News" поддерживает следующие функции:

#### 1. Получение курса валют

Чтобы получить курс валют, используйте следующую команду:

```
/getRate
```

Бот отправит вам текущий курс обмена для USD (доллар США), EUR (евро) и CNY (китайский юань) в российских рублях (RUB).

#### 2. Получение новостей

Чтобы получить последние новости, воспользуйтесь следующей командой:

```
/getNews
```

Бот загрузит новостные статьи с веб-сайта "https://wylsa.com/category/news/". Будут отображены до трех статей за один раз. Вы можете перемещаться по статьям с помощью предоставленных кнопок.

- Чтобы просмотреть следующий набор статей, нажмите кнопку "Далее".
- Чтобы просмотреть предыдущий набор статей, нажмите кнопку "Назад".
- Чтобы вернуться в главное меню, нажмите кнопку "Домой".

### Разработка

Если вы хотите изменить или внести свой вклад в развитие бота "Rate News", следуйте следующим шагам:

1. Клонируйте репозиторий.
2. Откройте проект в выбранной среде разработки.
3. Установите API-токен Telegram в поле `Config.TelegramApiToken` файла `Program.cs`. Получите API-токен, создав бота с помощью BotFather в Telegram.
4. Соберите и запустите проект.

### Зависимости

Бот "Rate News" создан с использованием следующих библиотек и фреймворков:

- `Telegram.Bot`: библиотека на C# для взаимодействия с API Telegram Bot.
- `HtmlAgilityPack`: библиотека на .NET для разбора HTML-документов.
- `System.Net.Http`: библиотека для отправки HTTP-запросов.
- `System.Threading

.Tasks`: библиотека для асинхронного программирования.
- `System.Xml`: библиотека для работы с XML-документами.
